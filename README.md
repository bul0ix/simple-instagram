# simple-instagram

This is a simple Instagram-like project.

## Features
- User authentication
- Image upload and sharing
- Comments and likes

## Installation
To set up the project, clone the repository:
```bash
git clone https://github.com/bul0ix/simple-instagram.git
```

## Usage
1. Navigate to the project directory.
2. Run the application with your preferred server setup.
3. Visit `http://localhost:3000` to access the app.
